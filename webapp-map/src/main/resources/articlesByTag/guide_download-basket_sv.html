Avgränsning av data

Öppna den data på kartan som du önskar ladda ner. Välj de objekt du vill ladda ner genom att avgränsa dem på kartan. Du avgränsa kartans data i helhet eller endast data från de områden du är intresserad av. Välj avgränsningsalternativet i ditt tycke. Avgränsningen tar med alla objekt som skär dvs. berör området i fråga (intersect). Du kan välja områden med olika avgränsningsalternativ ifall du trycker på ”flytta till nedladdningskorgen”-knappen.

Exempel.

Ifall du väljer som avgränsning kommunerna placeras kommungränserna på kartan. Klicka på de kommuner på kartan som du vill ladda ner data ifrån. De valda områdena blir orangea. Du kan ta bort ditt val genom att klicka på området igen eller genom att trycka på ”töm” då alla val försvinner.

Du kan även avgränsa fritt med rektangelverktyget. Klicka med den vänstra musknappen och dra ett område av valfri storlek på kartan.

Data till nedladdningskorgen.

När önskad data är avgränsad på kartan kan den flyttas till nedladdningskorgen. Tryck på ”Flytta till nedladdningskorgen” –knappen och vald data flyttas till följande flik i nedladdningskorgen.

Tjänsten visar dig hur många avgränsningar du gjort dvs. från hur många områden du kommer att ladda ned data ifrån. T.ex. Valda områden: 3 st., vill du flytta objekten till nedladdningskorgen?

Nedladdningskorgen

Data som flyttats till nedladdningskorgen dyker upp i fliken för nedladdningskorgen, varifrån du ser vilken data du är på väg att ladda ner. Tjänsten berättar även vilken licens användningen av vald data förutsätter.

Varje data som laddas ned är en egen helhet. Om du avgränsat flera områden på kartan kommer områdena att laddas ner som egna helheter. Du kan ännu vid den här punkten ta bort sådan data som du inte vill ladda ner eller gå tillbaka och lägga till mera data. När du är nöjd med den dataa som kommer att laddas ner, tryck på ”Nästa”

Att skicka data

Fyll i din e-postadress vid de ställen som den frågas efter. Nedladdad data kommer att skickas till dig via e-post. Du får via e-post en länk genom vilken du kommer åt att ladda ner din data. Det kan dröja en stund att skicka länken för nedladdning ifall din data är stor. Länken för nedladdning är i bruk i 10 dagar.

Vi sparar inte e-postadresserna. Adresserna behövs endast för nedladdning av din data.

OBS!

Ifall du stänger nedladdningskorgen utan att ladda ner data kan det hända att begränsningsfunktionen blir aktiv. Begränsningsfunktionen kan tas bort genom att trycka på ”Refresh” i webbläsaren.
